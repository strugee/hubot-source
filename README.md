# hubot-source

\"hubot: where's the source?\"

See [`src/source.coffee`](src/source.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-source --save`

Then add **hubot-source** to your `external-scripts.json`:

```json
[
  "hubot-source"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
